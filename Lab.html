2.Differentiation Tag

Tag

ความเหมือน

ความแตกต่าง
<hr> & <br> เป็นการขึ้นบรรทัดใหม่เหมือนกัน <hr> ขึ้นบรรทัดใหม่พร้อมเส้นแบ่ง horizontal
<br> เป็นการขึ้นบรรทัดใหม่ปกติ (ฺbreak tag)
<ul> & <ol> เป็นการ list เหมือนกัน <ul> เป็นการ list แบบ unorder list
<br> เป็นการ list แบบ orderlist
<select> & <checkbox> & <radio buton> เป็นการเลือกจาก list เหมือนกัน <select> เป็นการเลือกแบบ drop down list
<checkbox> เป็นการเลือกจากการติ๊ก กล่องข้อความได้หลายกล่องข้อความ
<radio buton> จะเลือกติ๊กกล่องข้อความได้แค่อันใดอันหนึ่ง
<div> & <span> เป็นการจัดกลุ่มข้อมูลเหมือนกัน <div> เป็นการจัดแบบ block element
<span> เป็นการจัดแบบ inline element
<script> & <style> เป็นการ code จากอีกpage บน html เหมือนกัน <script> เป็นการระบุ code ฝั่ง client-side script (เรียก page Javascript)
<style> เป็นการตกแต่งลักษณะต่างๆให้กับ HTML (เรียก page CSS style)
